# Deploying Stellar Router to Testnet

This guide walks you through deploying the **Stellar Router** contracts to Stellar Testnet.

## Prerequisites

- **Rust** (latest stable)
- **Soroban CLI** (recommended: `soroban-cli` v22.0.0 or newer)
- A Stellar testnet account (with some XLM)

### Install Soroban CLI

```bash
cargo install --locked soroban-cli
